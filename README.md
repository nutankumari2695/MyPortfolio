# Nutan Kumari - Portfolio Website

A modern, animated portfolio website built with Next.js 14, React, and Tailwind CSS.

## Features

- 🎨 **Stunning UI/UX**: Purple-blue gradient theme with glassmorphism effects
- ✨ **Smooth Animations**: Framer Motion powered animations and transitions
- 📱 **Fully Responsive**: Optimized for all devices (mobile, tablet, desktop)
- 🚀 **Modern Tech Stack**: Next.js 14, React 18, Tailwind CSS
- 🎯 **SEO Optimized**: Proper meta tags and semantic HTML
- 🌟 **Interactive Components**: Hover effects, typing animations, particle backgrounds

## Sections

1. **Hero** - Animated introduction with profile picture and social links
2. **About** - Personal summary with stats cards
3. **Skills** - Categorized technical skills with icons
4. **Experience** - Timeline of internship experiences
5. **Projects** - Showcase of 4 major projects
6. **Education** - Academic background timeline
7. **Contact** - Contact information and social media links

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **UI Library**: React 18
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Font**: Inter (Google Fonts)

## Project Structure

```
├── app/
│   ├── layout.js          # Root layout with metadata
│   ├── page.js            # Main page
│   └── globals.css        # Global styles
├── components/
│   ├── Navbar.jsx         # Navigation component
│   ├── Hero.jsx           # Hero section
│   ├── About.jsx          # About section
│   ├── Skills.jsx         # Skills section
│   ├── Experience.jsx     # Experience timeline
│   ├── Projects.jsx       # Projects showcase
│   ├── Education.jsx      # Education timeline
│   └── Contact.jsx        # Contact section
├── public/
│   └── profile.jpg        # Profile picture
└── package.json
```

## Customization

### Colors

Edit `tailwind.config.js` to customize the color scheme:
- Primary colors (purple tones)
- Accent colors (blue tones)

### Content

Update the content in each component file:
- Personal information in `Hero.jsx`
- Skills in `Skills.jsx`
- Projects in `Projects.jsx`
- Experience in `Experience.jsx`
- Education in `Education.jsx`
